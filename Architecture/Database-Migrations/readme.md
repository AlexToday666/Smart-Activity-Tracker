<h1 align="center">База данных и миграции</h1>

<h2 align="center">Flyway</h2>

Миграции находятся в `src/main/resources/db/migration` и применяются автоматически при запуске приложения.

Формат имени:

```text
V<версия>__<описание>.sql
```

<h2 align="center">Текущие миграции</h2>

- `V1__create_events.sql` — базовая таблица событий.
- `V2__project_scoped_events_and_api_keys.sql` — проекты, API-ключи, JSONB-метаданные, события, привязанные к проектам, и аналитические индексы.

<h2 align="center">Правило изменений схемы</h2>

Все изменения БД вносятся только через новые Flyway migrations. Ручные изменения схемы вне миграций не используются.
