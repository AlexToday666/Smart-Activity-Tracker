# База данных и миграции

## Flyway

Миграции находятся в `src/main/resources/db/migration` и применяются автоматически при запуске.

Формат имени:

```
V<версия>__<описание>.sql
```

## Текущая схема

`V1__create_events.sql` создаёт таблицу `events` и индексы по ключевым полям.

## Примечание по именам

В папке также присутствует `V1_create_events.sql`. По умолчанию Flyway применяет только файлы с двойным подчёркиванием, поэтому этот файл считается невалидным именем и игнорируется, если не переопределён паттерн миграций.
